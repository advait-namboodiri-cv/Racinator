# Racinator
An interactive Formula 1 analytics platform for exploring tracks, telemetry, and all round race strategy.

## Overview
This project is a full-stack F1 dashboard built to analyze race weekends and session data in a cleaner, more interactive way than raw timing tables. The long-term goal is to create a site where users can:

### Session Analytics
- lap time analysis
- weather timeline
- tire stint summaries

### Driver Comparison
- session-by-session performance
- consistency metrics

### Strategy Simulator
- tire degradation assumptions
- pit stop timing experiments
- what-if race scenarios
